# DSO_Email_OTP
This project is implemented through a RESTful Email OTP module using ASP.NET Core Web API and Entity Framework

It consists:
Generating OTP for valid enterprise emails
Verifying OTP with retry and expiry timeline

Only .dso.org.sg domain allowed
6-digit OTP generation
OTP is valid for 1 minute
Only latest OTP is valid
Max 10 attempts allowed
